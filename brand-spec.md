# Wechaser — Brand Spec (frozen 2026-05-04)

## Identity
- 品牌名: **WeChaser LLC** (英文为主，中文沿用 WeChaser，不用音译)
- Slogan (zh): 为制造企业部署一个真正能用的 AI 知识中枢
- Slogan (en): Deploy a working AI knowledge hub for manufacturers

## Audience
- 中高端制造企业 (中国大陆)，决策人为总经理 / IT负责人 / 数字化总监
- 双语切换：默认中文，提供英文 toggle（为未来海外预留）

## Visual direction
**东方留白派** — Anthropic × 无印良品 × 方正铸字。冷静、有重量、不吵闹。

### Color (oklch for harmonious math)
- Primary background: `oklch(0.97 0.012 75)` — 暖纸色 (#FAF7F2)
- Foreground / ink: `oklch(0.18 0.014 50)` — 深墨 (#211F1B)
- Accent (印章 seal red): `oklch(0.50 0.155 32)` — 朱砂 (#A8362C)
- Subtle line: `oklch(0.85 0.008 75)` — 浅墨线 (#D8D2C7)
- Highlight muted: `oklch(0.93 0.020 75)` — 卡片底 (#EFE9DD)

### Typography
- 中文标题: "Source Han Serif SC" / 思源宋体 (700, 900)
- 中文正文: "Source Han Sans SC" / 思源黑体 (400, 500)
- 英文标题: "Söhne" / 退而求其次 "Inter Display" semibold
- 英文正文: "Inter" 400/500
- 等宽 (技术注脚 / 数字): "JetBrains Mono"
- 字阶: 96 / 64 / 40 / 28 / 20 / 16 / 14 / 12

### Layout principles
- 12 列栅格，gutter 32px，max-width 1280px，section vertical rhythm 160px
- 单列叙述为主，双列对照，绝不用 3 卡片并排（避免 SaaS 模板感）
- 留白 > 内容，每屏只服务一个核心信息

### Iconography
- 不用 emoji
- 不用 3D render
- 用极细线条图（1.25px stroke）或印章红色块，少而精

### Brand seal element
- 朱砂红方形印章 含 "微" 字 / "WC" mono — 可作为页脚 stamp、CTA 落款、首屏右上角点睛

- ❌ Logo 源文件: 未提供，先用文字 logo + 印章 stamp 作为视觉签名

## Forbidden
- 紫色渐变（与现有 Lovable 模板撇清）
- 圆角卡片 + 左侧色条 + emoji icon 三件套
- "Powered by AI" / "革命性" / "颠覆" 之类陈词
- 满屏 hero 视频或 Three.js 粒子背景
